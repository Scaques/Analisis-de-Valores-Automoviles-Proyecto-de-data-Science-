# Analisis-de-Valores-Automoviles-Proyecto-de-data-Science-
Pipeline de Data Science en Python para el mercado automotriz chileno. Incluye automatización de datos, limpieza avanzada con Pandas y un modelo de Machine Learning (Random Forest) que tasa vehículos según su año, kilometraje y marca. Diseñado con arquitectura limpia y Pipelines de Scikit-Learn. 🚀
